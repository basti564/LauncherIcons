# LauncherIcons
Repo for Pico, Oculus, Viveport and Vive Business Launcher Icons
## Supported Formats
### Pico
- pico_landscape
- pico_square
### Oculus
- oculus_icon
- oculus_landscape
- oculus_square
- oculus_portrait
### Viveport
- viveport_small
- viveport_medium
- viveport_large
- viveport_square
### Vive Business
- vive_business_small
- vive_business_medium
- vive_business_large
- vive_business_square
## Planned Features
- download cover images for SideQuest apps (implemented, but currently not enabled)
